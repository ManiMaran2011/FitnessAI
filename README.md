# FitnessAI

A simple AI project to recommend workouts and meals based on user input.

## Files
- `app.py` : Main Python source code
- `Instructions.docx` : User guide and instructions

## How to Run
1. Install Python 3.x
2. Install required libraries (`pip install -r requirements.txt` if you have them)
3. Run the script: `python app.py`
